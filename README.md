# Análisis de Adjetivos Musicales

Aplicación web para analizar adjetivos en textos musicales.

## Características

- Carga archivos .txt y .pdf
- Clasifica adjetivos musicales y físicos
- Genera reportes en PDF y CSV
- Gráficos interactivos

## Uso

Solo tienes que subir tus archivos y buscar un nombre.
